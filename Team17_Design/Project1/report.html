Short Analysis Paragraph

The ChocAn System includes hardware for user interfaces that interact with the computerized processing software at the ChocAn Data Center. Providers, managers, and operators at the Data Center interact with individual terminals that include a keyboard and a screen. In order for these terminals to access the ChocAn data files stored on disks, the processing software validates each user and reflects that computerized validation on their specific terminal. The ChocAn system also interacts with Acme Accounting Services computers and banking computers. Additionally, the system is connected to a timer that automatically triggers the main accounting procedure. Other computerized processes include printing/sending reports as email attachments, totaling fees, and the validation process for the users.

Use case: Validates Member

Context: In order to provide service, provider must swipe card and confirm the member exists and is up to date with payments

Actors: Provider

Main Success Scenario:

1. Provider swipes patient's card

2. Terminal communicates with ChocAn data center and confirms that the member exists and is up to date financially

3. Terminal displays "Validated"

Extensions:

3a. Patient's card is not recognized and screen displays "Invalid Number"

3b. Patient is not up to date on payments and screen displays "Member suspended"

Use Case: Bills ChocAn for service

Actors: Provider

Context: After an appointment happens, the provider records the appointment into ChocAn services for eventual reimbursement

Main Success Scenario:

1. Provider does the Validates Member subprocess

2. Provider enters service date in the format MM-DD-YYYY

3. Provider searches provider directory for service code

4. Software validates service code and presents to provider

5. Provider enters comments about service provided

6. Software writes to disk a record with the following information

 6a. Current date and time

 6b. Date service was provided

 6c. Provider number

 6d. Service code

 6e. Comments (100 characters, optional)

7. Software displays correct fee to be paid to provider

Extensions:

4a. If service code invalid, display error message and prompt retype code

5. Updated Glossary

Alphabetized

  • EFT Data- Electronic funds transfer data which is used by the banking software to record and manage all funds transferred through ChocAn

  • EFT Report - Report for banking software to perform financial transfers to providers

  • Manager - ChocAn senior employee who is able to create weekly summaries

  • Member - Paying subscriber to ChocAn who is up to date on payments

  • Member Report - Report to member summarizing account as well as each appointment from that week, with the following information:
    Member number (9 digits).
    Member street address (25 characters).
    Member city (14 characters).
    Member state (2 letters).
    Member ZIP code (5 digits).
    For each service provided, the following details are required:
    - Date of service (MM–DD–YYYY).
    - Provider name (25 characters).
    - Service name (20 characters).

  • Operator - ChocAn employee who is able to edit records

  • Plastic Card - Unique ID card given to all members that is used in validation

  • Provider - Health care professional who provides ChocAn services

  • Provider Directory - File containing all possible service ID codes in alphabetical order

  • Provider Number - Unique number each provider enters to turn on their terminal

  • Provider Report - Report of all ChocAn services provided to members that week, which should match the provider's own records. Included is the following information:
    Provider name (25 characters).
    Provider number (9 digits).
    Provider street address (25 characters).
    Provider city (14 characters).
    Provider state (2 letters).
    Provider ZIP code (5 digits).
    For each service provided, the following details are required:
    Date of service (MM–DD–YYYY).
    Date and time data were received by the computer (MM–DD–YYYY HH:MM:SS).
    Member name (25 characters).
    Member number (9 digits).
    Service code (6 digits).
    Fee to be paid (up to $999.99).
    Total number of consultations with members (3 digits).
    Total fee for week (up to $99,999.99).

  • Service code - six-digit code corresponding to a service used in the Provider Directory

  • Summary report- Weekly report sent to a ChocAn manager which lists all provider activities, including sessions and services, and the amount providers are owed

  • Suspended Member - Member who has missed a payment but has not terminated their membership

  • Terminal - The device that the health care professional uses to access ChocAn software

  • Timer - Clock device that automatically runs the main accounting procedure on Fridays at 5pm

  • Weekly report- A financial/order history statement sent to all ChocAn members each week
